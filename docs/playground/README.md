# PDF playground WASM artifact (generated)

`markdownpdf.wasm.gz` is a gzipped WebAssembly build of `MarkdownPDFPlaygroundCLI`
(the MarkdownPDF engine + MathTypeset), Markdown in -> PDF out, run in the browser.
Single-commit artifact branch; do not base work on it. The tiledown.com PDF
playground fetches this file.
