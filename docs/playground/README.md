# Playground WASM artifact (generated)

`tiledown-math.wasm.gz` is a gzipped WebAssembly build of `MathPlaygroundCLI`
(the MathTypeset + TileMath math engine), rebuilt by the Playground WASM
workflow on every engine change. This branch is a single-commit artifact ref;
do not base work on it. The tiledown.com playground fetches this file.
