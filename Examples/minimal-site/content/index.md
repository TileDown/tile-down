# Minimal Site

A compact Tiledown demo built from Markdown, a built-in layout, and the `system`
theme.

The site has the pieces a small personal site needs: an about page, a contact
page, three posts, social links in the footer, and an RSS feed.

## Start with content

Pages live in folders named with their slug. Each folder has an `index.md` file.
The build turns `content/about/index.md` into `/about/` and
`content/posts/designing-from-system-defaults/index.md` into
`/posts/designing-from-system-defaults/`.

## Built with TileKit

The footer link points to TileKit, the Swift engine inside Tiledown that renders
the site.
