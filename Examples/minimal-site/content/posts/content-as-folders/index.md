# Content as folders

Tiledown discovers `index.md` files below the content root. The folder path is the
slug, so a post at `content/posts/content-as-folders/index.md` becomes
`/posts/content-as-folders/`.

That makes URLs visible in the file tree and keeps each page ready for page-local
assets later.
